# KoHack26-TABC1

## Files
1. Tehillim_Matcher.java — The main file, and the only one to be executed, providing the interface in terminal.
2. tehillim.xml — A list of String resources, containing all the Prakim of tehillim in english, hebrew, and transliterated form, for the Tehillim_Matcher.java file to pull on.
3. People_Table.sql — The table schematic for the sql database that is employed.

## Features
1. Allows users to enter people who need tehillim said for them (Tehillim_Matcher.java)
2. Gives users random people who need tehillim said (Tehillim_Matcher.java)
3. Provides language options between english, hebrew, and hebrew transliteration for tehillim (Tehillim_Matcher.java)
